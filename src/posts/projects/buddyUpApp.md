## Buddy App
A Flask application allow uses to create a task and appoint a buddy which reminders are sent to, to remind the task creator.

#### Technology Used 
- Flask
- SQLite
- Docker
- Bootstrap (HTML and CSS)
- Jinja Templating

### Features 
- Create Tasks and set due dates.
